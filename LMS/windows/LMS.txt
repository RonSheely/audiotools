			Level Measuring Set
			===================

Audio level measuring set. Measures frequency and relative level with
a spectrum display. Absolute level is not possible because of all the
variables in the sound system.

Using
=====

The measuring set displays frequency and relative level with a
spectrum display. Input is from the currently selected audio input.

========

Use the makefile with Mingw or Cygwin GCC, or use a Microsoft C
compiler to build.
